jscheme
=======

JScheme: evaluating Scheme(R4RS) expressions from java

# How to use?

```
 Scheme  scheme = new Scheme();
 Object result = scheme.eval(new StringReader("(+ 1 (+ 2 3))"));
```

# Jscheme History
This is the code found at http://norvig.com/jscheme.html by Peter Norvig.
